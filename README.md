# 🎀 My Photo Booth - Cute Digital Photo Booth

A dreamy photo booth web app, built for capturing fun moments instantly, no setup, no complexity.

**Live Demo:** https://my-photobooth-web.vercel.app/

---

## Features
- Live camera with 16 filters
- 3 or 4 photo strips
- Frame styles (white, pink, cream, lavender)
- Custom captions + date stamp
- Download as JPEG, PNG, or Instagram Story (9:16)
- Native share support (mobile)
- 100% client-side, no backend needed

---

## Filters

Each filter is designed to feel **visually distinct**:

* Natural (no filter)
* 90s Film
* Neon Pop
* B&W Classic
* Golden Hour
* Disposable Flash
* Dreamy Blur
