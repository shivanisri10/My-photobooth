# 🎀 My Photo Booth - Cute Digital Photo Booth

A dreamy, blush-pink Korean-inspired photo booth web app.

## Features
- Live camera with 16 filters
- 3 or 4 photo strips
- Frame styles (white, pink, cream, lavender)
- Custom captions + date stamp
- Download as JPEG, PNG, or Instagram Story (9:16)
- Native share support (mobile)
- 100% client-side, no backend needed

## Deploy to Vercel (Free)

### Option A: Deploy via GitHub (Recommended)

1. Push this project to a GitHub repo
2. Go to https://vercel.com and sign in with GitHub
3. Click "New Project"
4. Import your repo
5. Click "Deploy" — done!

### Option B: Deploy via Vercel CLI

```bash
npm install
npx vercel --prod
```

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000
